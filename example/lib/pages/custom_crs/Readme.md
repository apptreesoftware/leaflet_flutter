# Custom CRS

## Proj4Crs

Proj4Crs extends Crs. Proj4Crs can be created with a factory with the following parameters:

- `code`: string identifier for the selected CRS, e.g. 'EPSG:3413'.
- `proj4Projection`: with `proj4dart`
- `resolutions`
- `bounds`
- `origins`
- `scales`
- `transformation`
